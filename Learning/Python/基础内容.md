# 基础内容

计算机的本质工作原理就是：

数据和指令存储在硬盘中。然后运行的时候从硬盘中读取内容到内存，再将指令载入cpu进行运算，从内存或者硬盘中获取参加运算的数据。然后将运算结果放回内存或者硬盘。在这期间可以通过指令让机器发出声音、显示图像等等。其实都是数据的可知觉化显示。

一台计算机的本质其实和人一样，具备基础的运算能力，遵循基本的运算规则。然后对输入的信息加工，辅之以特殊的规则，从而得到"有意义"的数据。
